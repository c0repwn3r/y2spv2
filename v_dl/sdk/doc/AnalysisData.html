Class: AnalysisData

AnalysisData

new AnalysisData()

AnalysisData structure, used as a container for all face analysis results. This structure is passed as a parameter to the VisageFaceAnalyser.analyseImage() or VisageFaceAnalyser.analyseStream() method. Both methods store analysis results into the structure.

If an operation was not performed (turned off via options) or failed, the respective validity boolean will return false and estimations will contain the default value.
